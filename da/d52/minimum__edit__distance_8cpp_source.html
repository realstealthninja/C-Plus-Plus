1/**
2 * @file
3 * @brief Implementation of [Minimum Edit Distance](https://en.wikipedia.org/wiki/Edit_distance) using Dynamic Programing
4 *
5 * @details
6 *
7 * Given two strings str1 & str2 and we have to calculate the minimum
8 * number of operations (Insert, Remove, Replace) required to convert
9 * str1 to str2.
10 *
11 * ### Algorithm
12 *
13 * We will solve this problem using Naive recursion. But as we are
14 * approaching with a DP solution. So, we will take a DP array to
15 * store the solution of all sub-problems so that we don't have to
16 * perform recursion again and again. Now to solve the problem, We
17 * can traverse all characters from either right side of the strings
18 * or left side. Suppose we will do it from the right side. So, there
19 * are two possibilities for every pair of characters being traversed.
20 * 1. If the last characters of two strings are the same, Ignore
21 * the characters and get the count for the remaining string.
22 * So, we get the solution for lengths m-1 and n-1 in a DP array.
23 *
24 * 2. Else, (If last characters are not the same), we will consider all
25 * three operations (Insert, Remove, Replace) on the last character of
26 * the first string and compute the minimum cost for all three operations
27 * and take the minimum of three values in the DP array.
28 * For Insert: Recur for m and n-1
29 * For Remove: Recur for for m-1 and n
30 * For Replace: Recur for for m-1 and n-1
31 *
32 * @author [Nirjas Jakilim](github.com/nirzak)
33 */

52/**
53 * @brief Takes input of the cost of
54 * three operations: Insert, Replace and Delete
55 * and return the minimum cost among them.
56 * @param x used to pass minimum cost of Insert operations
57 * @param y used to pass minimum cost of Replace operations
58 * @param z used to pass minimum cost of Delete operations
59 * @returns x if `x` is the minimum value
60 * @returns y if `y` is the minimum value
61 * @returns z if `z` is the minimum value
62 */
63uint64_t min(uint64_t x, uint64_t y, uint64_t z) {
64 if (x <= y && x <= z) {
65 return x; /// returns x, if x is the minimum value
66 }
67 if (y <= x && y <= z) {
68 return y; /// returns y, if y is the minimum value
69 }
70 else {
71 return z; /// returns z if z is the minimum value
72 }
73}

75/**
76 * @brief Calculates and stores the result
77 * of all the sub-problems, so that we don't have to recur to compute
78 * the minimum cost of a particular operation if it is already
79 * computed and stored in the `dp` vector.
80 * @param dp vector to store the computed minimum costs
81 * @param str1 to pass the 1st string
82 * @param str2 to pass the 2nd string
83 * @param m the length of str1
84 * @param n the length of str2
85 * @returns dp[m][n] the minimum cost of operations
86 * needed to convert str1 to str2
87 */
88uint64_t editDistDP(std::string str1, std::string str2, uint64_t m, uint64_t n) {
89 /// Create a table to store results of subproblems
90 std::vector<std::vector<uint64_t>>dp(m+1, std::vector<uint64_t>(n+1)); /// creasting 2D vector dp to store the results of subproblems
91
92 /// Fill d[][] in bottom up manner
93 for (uint64_t i = 0; i <= m; i++) {
94 for (uint64_t j = 0; j <= n; j++) {
95 /// If first string is empty, only option is to
96 /// insert all characters of second string
97 if (i == 0) {
98 dp[i][j] = j; /// Minimum operations = j
99 }
100
101 /// If second string is empty, only option is to
102 /// remove all characters of second string
103 else if (j == 0) {
104 dp[i][j] = i; /// Minimum operations = i
105 }
106
107 /// If last characters are same, ignore last char
108 /// and recur for remaining string
109 else if (str1[i - 1] == str2[j - 1]) {
110 dp[i][j] = dp[i - 1][j - 1];
111 }
112
113 /// If the last character is different, consider all
114 /// possibilities and find the minimum
115 else {
116 dp[i][j] = 1 + min(dp[i][j - 1], // Insert
117 dp[i - 1][j], // Remove
118 dp[i - 1][j - 1]); // Replace
119 }
120 }
121 }
122
123 return dp[m][n]; /// returning the minimum cost of operations needed to convert str1 to str2
124}

11 * \details
12 * Breadth First Search also quoted as BFS is a Graph Traversal Algorithm.
13 * Time Complexity O(|V| + |E|) where V are the number of vertices and E
14 * are the number of edges in the graph.
15 *
16 * Applications of Breadth First Search are
17 *
18 * 1. Finding shortest path between two vertices say u and v, with path
19 * length measured by number of edges (an advantage over depth first
20 * search algorithm)
21 * 2. Ford-Fulkerson Method for computing the maximum flow in a flow network.
22 * 3. Testing bipartiteness of a graph.
23 * 4. Cheney's Algorithm, Copying garbage collection.
24 *
25 * And there are many more...
26 *
27 * <h4>working</h4>
28 * In the implementation below we first created a graph using the adjacency
29 * list representation of graph.
30 * Breadth First Search Works as follows
31 * it requires a vertex as a start vertex, Start vertex is that vertex
32 * from where you want to start traversing the graph.
33 * We maintain a bool array or a vector to keep track of the vertices
34 * which we have visited so that we do not traverse the visited vertices
35 * again and again and eventually fall into an infinite loop. Along with this
36 * boolen array we use a Queue.
37 *
38 * 1. First we mark the start vertex as visited.
39 * 2. Push this visited vertex in the Queue.
40 * 3. while the queue is not empty we repeat the following steps
41 *
42 * 1. Take out an element from the front of queue
43 * 2. Explore the adjacency list of this vertex
44 * if element in the adjacency list is not visited then we
45 * push that element into the queue and mark this as visited
46 *
47 */

91 /**
92 * this function performs the breadth first search on graph and return a
93 * mapping which maps the nodes to a boolean value representing whether the
94 * node was traversed or not.
95 */
96 std::map<T, bool> breadth_first_search(T src) {
97 /// mapping to keep track of all visited nodes
98 std::map<T, bool> visited;
99 /// initialise every possible vertex to map to false
100 /// initially none of the vertices are unvisited
101 for (auto const &adjlist : adjacency_list) {
102 visited[adjlist.first] = false;
103 for (auto const &node : adjacency_list[adjlist.first]) {
104 visited[node] = false;
105 }
106 }
107
108 /// queue to store the nodes which are yet to be traversed
109 std::queue<T> tracker;
110
111 /// push the source vertex to queue to begin traversing
112 tracker.push(src);
113 /// mark the source vertex as visited
114 visited[src] = true;
115 while (!tracker.empty()) {
116 /// traverse the graph till no connected vertex are left
117 /// extract a node from queue for further traversal
118 T node = tracker.front();
119 /// remove the node from the queue
120 tracker.pop();
121 for (T const &neighbour : adjacency_list[node]) {
122 /// check every vertex connected to the node which are still
123 /// unvisited
124 if (!visited[neighbour]) {
125 /// if the neighbour is unvisited , push it into the queue
126 tracker.push(neighbour);
127 /// mark the neighbour as visited
128 visited[neighbour] = true;
129 }
130 }
131 }
132 return visited;
133 }

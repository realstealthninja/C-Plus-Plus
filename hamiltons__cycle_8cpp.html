Detailed Description

The implementation of Hamilton's cycle dynamic solution for vertices number less than 20.

I use matrix and for every ( and ) in the matrix I store true if it is possible to get to all vertices on which position in i's binary representation is 1 so as would be the last one.

In the the end if any cell in row is true there exists hamiltonian cycle.

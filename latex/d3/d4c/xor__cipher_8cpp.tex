\doxysubsection{Detailed Description}
Implementation of \href{https://en.wikipedia.org/wiki/XOR_cipher}{\texttt{ XOR cipher}} algorithm. 

In cryptography, the simple \doxylink{namespace_x_o_r}{XOR} cipher is a type of additive cipher, an encryption algorithm that operates according to the principles\+:


\begin{DoxyItemize}
\item $A {\oplus} 0 = A$
\item $A {\oplus} A = 0$
\item $ (A {\oplus} B) {\oplus} C = A {\oplus} (B {\oplus} C)$
\item $ (B {\oplus} A) {\oplus} B = B {\oplus} 0 = B $
\end{DoxyItemize}

where $\oplus$ symbol denotes the exclusive disjunction (\doxylink{namespace_x_o_r}{XOR}) operation. This operation is sometimes called modulus 2 addition (or subtraction, which is identical). With this logic, a string of text can be encrypted by applying the bitwise \doxylink{namespace_x_o_r}{XOR} operator to every character using a given key. To decrypt the output, merely reapplying the \doxylink{namespace_x_o_r}{XOR} function with the key will remove the cipher.\hypertarget{xor__cipher_8cpp_autotoc_md3}{}\doxysubsubsubsection{\texorpdfstring{Algorithm}{Algorithm}}\label{xor__cipher_8cpp_autotoc_md3}
Choose the key for encryption and apply \doxylink{namespace_x_o_r}{XOR} operation to each character of a string. Reapplying \doxylink{namespace_x_o_r}{XOR} operation to each character of encrypted string will give original string back.

\begin{DoxyNote}{Note}
This program implements \doxylink{namespace_x_o_r}{XOR} Cipher for string with ASCII characters.
\end{DoxyNote}

\doxysubsection{Detailed Description}
The \href{https://cp-algorithms.com/string/z-function.html}{\texttt{ Z function}} for finding occurences of a pattern within a piece of text with time and space complexity O(n + m) 


\begin{DoxyEnumerate}
\item The Z-\/function for a string is an array of length n where the i-\/th element is equal to the greatest number of characters starting from the position i that coincide with the first characters of s.
\item E.\+g.\+: string\+: ababb then z\mbox{[}2\mbox{]}=2 as s\mbox{[}2\mbox{]}=s\mbox{[}0\mbox{]} and s\mbox{[}3\mbox{]}=s\mbox{[}1\mbox{]} and s\mbox{[}4\mbox{]}!=s\mbox{[}2\mbox{]} \begin{DoxyAuthor}{Author}

\end{DoxyEnumerate}

\doxysubsection{Detailed Description}
Check if a directed graph has a cycle or not.

This class provides 2 methods to check for cycle in a directed graph\+: is\+Cyclic\+DFS \& is\+Cyclic\+BFS.


\begin{DoxyItemize}
\item is\+Cyclic\+DFS uses DFS traversal method to check for cycle in a graph.
\item is\+Cyclid\+BFS used BFS traversal method to check for cycle in a graph. 
\end{DoxyItemize}

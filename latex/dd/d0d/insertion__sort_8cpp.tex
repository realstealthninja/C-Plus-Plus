\doxysubsection{Detailed Description}
\href{https://en.wikipedia.org/wiki/Insertion_sort}{\texttt{ Insertion Sort Algorithm (Insertion Sort)}} 

Insertion sort is a simple sorting algorithm that builds the final sorted array one at a time. It is much less efficient compared to other sorting algorithms like heap sort, merge sort or quick sort. However it has several advantages such as
\begin{DoxyEnumerate}
\item Easy to implement
\item For small set of data it is quite efficient
\item More efficient that other Quadratic complexity algorithms like Selection sort or bubble sort.
\item It\textquotesingle{}s stable that is it does not change the relative order of elements with equal keys
\item Works on hand means it can sort the array or list as it receives.
\end{DoxyEnumerate}

It is based on the same idea that people use to sort the playing cards in their hands. the algorithms goes in the manner that we start iterating over the array of elements as soon as we find a unsorted element that is a misplaced element we place it at a sorted position.

Example execution steps\+:
\begin{DoxyEnumerate}
\item Suppose initially we have \begin{bmatrix}4 &3 &2 &5 &1\end{bmatrix}
\item We start traversing from 4 till we reach 1 when we reach at 3 we find that it is misplaced so we take 3 and place it at a correct position thus the array will become \begin{bmatrix}3 &4 &2 &5 &1\end{bmatrix}
\item In the next iteration we are at 2 we find that this is also misplaced so we place it at the correct sorted position thus the array in this iteration becomes \begin{bmatrix}2 &3 &4 &5 &1\end{bmatrix}
\item We do not do anything with 5 and move on to the next iteration and select 1 which is misplaced and place it at correct position. Thus, we have \begin{bmatrix}1 &2 &3 &4 &5\end{bmatrix} 
\end{DoxyEnumerate}

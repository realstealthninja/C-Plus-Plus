\doxysubsection{Detailed Description}
The implementation of \href{https://en.wikipedia.org/wiki/Hamiltonian_path}{\texttt{ Hamilton\textquotesingle{}s cycle}} dynamic solution for vertices number less than 20. 

I use $2^n\times n$ matrix and for every $[i, j]$ ( $i < 2^n$ and $j < n$) in the matrix I store {\ttfamily true} if it is possible to get to all vertices on which position in {\ttfamily i}\textquotesingle{}s binary representation is {\ttfamily 1} so as $j$ would be the last one.

In the the end if any cell in $(2^n - 1)^{\mbox{th}}$ row is {\ttfamily true} there exists hamiltonian cycle.

\doxysubsection{Detailed Description}
A simple \href{https://en.wikipedia.org/wiki/Matching_game}{\texttt{ Memory Game}} with {\bfseries{3 different sizes}} and multiple letters. 

The game consists on finding {\bfseries{the pair}} of all the given letters depending on the table size. Once all of the instances are all found, the game will end and will ask you if you\textquotesingle{}d like to play again or not.

It provides {\bfseries{3 different sizes}} available that the user can choose (4x2, 5x2, 7x2). 7x2 being the biggest table size and hardest mode. The bigger the size, {\bfseries{the more letters are available}}.

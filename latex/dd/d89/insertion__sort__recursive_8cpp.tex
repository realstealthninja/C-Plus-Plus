\doxysubsection{Detailed Description}
Insertion Sort Algorithm. 

\begin{DoxyAuthor}{Author}
\href{https://github.com/Fandroid745}{\texttt{ Dhanush S}}
\end{DoxyAuthor}
Insertion sort is a simple sorting algorithm that builds the final sorted array one element at a time. It is much less efficient compared to other sorting algorithms like heap sort, merge sort, or quick sort.

However, it has several advantages\+:
\begin{DoxyItemize}
\item Easy to implement.
\item Efficient for small data sets.
\item More efficient than other O(nÂ²) algorithms like selection sort or bubble sort.
\item Stable\+: it does not change the relative order of elements with equal keys.
\end{DoxyItemize}

Insertion sort works similarly to how people sort playing cards in their hands. The algorithm iterates through the list and inserts each element into its correct position in the sorted portion of the array.

The time complexity of the algorithm is $O(n^2)$, and in some cases, it can be $O(n)$.

Example execution\+:
\begin{DoxyEnumerate}
\item Start with the array \mbox{[}4, 3, 2, 5, 1\mbox{]}.
\item Insert 3 in its correct position\+: \mbox{[}3, 4, 2, 5, 1\mbox{]}.
\item Insert 2\+: \mbox{[}2, 3, 4, 5, 1\mbox{]}.
\item Continue this until the array is sorted\+: \mbox{[}1, 2, 3, 4, 5\mbox{]}. 
\end{DoxyEnumerate}

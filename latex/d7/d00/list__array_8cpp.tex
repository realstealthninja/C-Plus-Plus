\doxysubsection{Detailed Description}
\href{https://en.wikipedia.org/wiki/Dynamic_array}{\texttt{ Dynamic Array}} 

The \doxylink{namespacelist__array}{list\+\_\+array} is the implementation of list represented using array. We can perform basic CRUD operations as well as other operations like sorting etc.\hypertarget{list__array_8cpp_autotoc_md42}{}\doxysubsubsubsection{\texorpdfstring{Algorithm}{Algorithm}}\label{list__array_8cpp_autotoc_md42}
It implements various method like insert, sort, search etc. efficiently. You can select the operation and methods will do the rest work for you. You can insert element, sort them in order, search efficiently, delete values and print the list.

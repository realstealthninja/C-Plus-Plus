\doxysubsection{Detailed Description}
\href{https://en.wikipedia.org/wiki/Sieve_of_Eratosthenes}{\texttt{ The Sieve of Eratosthenes}} 

Store an array of booleans where a true value indicates that it\textquotesingle{}s index is prime. For all the values in the array starting from 2 which we know is prime, we walk the array in multiples of the current outer value setting them to not prime. If we remove all multiples of a value as we see it, we\textquotesingle{}ll be left with just primes.

Pass "{}print"{} as a command line arg to see the generated list of primes \begin{DoxyAuthor}{Author}

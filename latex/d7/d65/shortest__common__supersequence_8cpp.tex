\doxysubsection{Detailed Description}
SCS is a string Z which is the shortest supersequence of strings X and Y (may not be continuous in Z, but order is maintained). 

The idea is to use lookup table method as used in LCS. For example\+: example 1\+:-\/ X\+: \textquotesingle{}ABCXYZ\textquotesingle{}, Y\+: \textquotesingle{}ABZ\textquotesingle{} then Z will be \textquotesingle{}ABCXYZ\textquotesingle{} (y is not continuous but in order)

For example\+: example 2\+:-\/ X\+: \textquotesingle{}AGGTAB\textquotesingle{}, Y\+: \textquotesingle{}GXTXAYB\textquotesingle{} then Z will be \textquotesingle{}AGGXTXAYB\textquotesingle{} \begin{DoxyAuthor}{Author}

\end{DoxyAuthor}

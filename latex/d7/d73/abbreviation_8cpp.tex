\doxysubsection{Detailed Description}
Implementation of \href{https://www.hackerrank.com/challenges/abbr/problem}{\texttt{ Abbrievation}} 

Given two strings, {\ttfamily a} and {\ttfamily b}, determine if it\textquotesingle{}s possible to make {\ttfamily a} equal to {\ttfamily b} You can perform the following operations on the string {\ttfamily a}\+:
\begin{DoxyEnumerate}
\item Capitalize zero or more of {\ttfamily a}\textquotesingle{}s lowercase letters.
\item Delete all of the remaining lowercase letters in {\ttfamily a}.
\end{DoxyEnumerate}\hypertarget{abbreviation_8cpp_autotoc_md70}{}\doxysubsubsubsection{\texorpdfstring{Algorithm}{Algorithm}}\label{abbreviation_8cpp_autotoc_md70}
The idea is in the problem statement itself\+: iterate through characters of string {\ttfamily a} and {\ttfamily b} (for character indexes {\ttfamily i} and {\ttfamily j} respectively)\+:
\begin{DoxyEnumerate}
\item If {\ttfamily a\mbox{[}i\mbox{]}} and {\ttfamily b\mbox{[}j\mbox{]}} are equal, then move to next position
\item If {\ttfamily a\mbox{[}i\mbox{]}} is lowercase of {\ttfamily b\mbox{[}j\mbox{]}}, then explore two possibilities\+: a. Capitalize {\ttfamily a\mbox{[}i\mbox{]}} or b. Skip {\ttfamily a\mbox{[}i\mbox{]}}
\item If the {\ttfamily a\mbox{[}i\mbox{]}} is not uppercase, just discard that character, else return {\ttfamily false}
\end{DoxyEnumerate}

Time Complexity\+: (O(\texorpdfstring{$\vert$}{|}a\texorpdfstring{$\vert$}{|}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\vert$}{|}b\texorpdfstring{$\vert$}{|})) where {\ttfamily \texorpdfstring{$\vert$}{|}a\texorpdfstring{$\vert$}{|}} =\texorpdfstring{$>$}{>} length of string {\ttfamily a} \begin{DoxyAuthor}{Author}
\href{https://github.com/AshishYUO}{\texttt{ Ashish Daulatabad}} 
\end{DoxyAuthor}

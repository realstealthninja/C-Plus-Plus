\doxysubsection{Detailed Description}
Matrix Exponentiation. 

The problem can be solved with DP but constraints are high. ~\newline
 $a_i = b_i$ (for $i <= k$) ~\newline
 $a_i = c_1 a_{i-1} + c_2 a_{i-2} + ... + c_k a_{i-k}$ (for $i > k$) ~\newline
Taking the example of Fibonacci series, $k=2$ ~\newline
 $b_1 = 1,\; b_2=1$ ~\newline
 $c_1 = 1,\; c_2=1$ ~\newline
 $a = \begin{bmatrix}0& 1& 1& 2& \ldots\end{bmatrix}$ ~\newline
This way you can find the $10^{18}$ fibonacci numberMOD. I have given a general way to use it. The program takes the input of B and C matrix.

Steps for Matrix Expo
\begin{DoxyEnumerate}
\item Create vector F1 \+: which is the copy of B.
\item Create transpose matrix (Learn more about it on the internet)
\item Perform $T^{n-1}$ \mbox{[}transpose matrix to the power n-\/1\mbox{]}
\item Multiply with F to get the last matrix of size (1 $\times$k).
\end{DoxyEnumerate}

The first element of this matrix is the required result.

\doxysubsection{Detailed Description}
\mbox{[}Bidirectional Dijkstra Shortest Path Algorithm\mbox{]} (\href{https://www.coursera.org/learn/algorithms-on-graphs/lecture/7ml18/bidirectional-dijkstra}{\texttt{ https\+://www.\+coursera.\+org/learn/algorithms-\/on-\/graphs/lecture/7ml18/bidirectional-\/dijkstra}}) 

\begin{DoxyAuthor}{Author}
\href{http://github.com/Marinovsky}{\texttt{ Marinovksy}}
\end{DoxyAuthor}
This is basically the same Dijkstra Algorithm but faster because it goes from the source to the target and from target to the source and stops when finding a vertex visited already by the direct search or the reverse one. Here some simulations of it\+: \href{https://www.youtube.com/watch?v=DINCL5cd_w0&t=24s}{\texttt{ https\+://www.\+youtube.\+com/watch?v=\+DINCL5cd\+\_\+w0\&t=24s}}

\doxysubsection{Detailed Description}
Implementation of \href{https://en.wikipedia.org/wiki/Euler\%27s_totient_function}{\texttt{ Euler\textquotesingle{}s Totient}} @description Euler Totient Function is also known as phi function. 

\[\phi(n) =
\phi\left({p_1}^{a_1}\right)\cdot\phi\left({p_2}^{a_2}\right)\ldots\] where $p_1$, $p_2$, $\ldots$ are prime factors of n. ~\newline
3 Euler\textquotesingle{}s properties\+:
\begin{DoxyEnumerate}
\item $\phi(n) = n-1$
\item $\phi(n^k) = n^k - n^{k-1}$
\item $\phi(a,b) = \phi(a)\cdot\phi(b)$ where a and b are relative primes.
\end{DoxyEnumerate}

Applying this 3 properties on the first equation.  \[\phi(n) =
n\cdot\left(1-\frac{1}{p_1}\right)\cdot\left(1-\frac{1}{p_2}\right)\cdots\] where $p_1$, $p_2$... are prime factors. Hence Implementation in $O\left(\sqrt{n}\right)$. ~\newline
Some known values are\+:
\begin{DoxyItemize}
\item $\phi(100) = 40$
\item $\phi(1) = 1$
\item $\phi(17501) = 15120$
\item $\phi(1420) = 560$ \begin{DoxyAuthor}{Author}
\href{https://github.com/mann2108}{\texttt{ Mann Mehta}} 
\end{DoxyAuthor}

\end{DoxyItemize}

Depth First Search also quoted as DFS is a \doxylink{class_graph}{Graph} Traversal Algorithm. Time Complexity O(\texorpdfstring{$\vert$}{|}\+V\texorpdfstring{$\vert$}{|} + \texorpdfstring{$\vert$}{|}\+E\texorpdfstring{$\vert$}{|}) where V is number of vertices and E is number of edges in graph.

Application of Depth First Search are


\begin{DoxyEnumerate}
\item Finding connected components
\item Finding 2-\/(edge or vertex)-\/connected components.
\item Finding 3-\/(edge or vertex)-\/connected components.
\item Finding the bridges of a graph.
\item Generating words in order to plot the limit set of a group.
\item Finding strongly connected components.
\end{DoxyEnumerate}

\doxysubsubsubsubsection*{Working}


\begin{DoxyEnumerate}
\item Mark all vertices as unvisited (colour it WHITE).
\item Push starting vertex into the stack and colour it GREY.
\item Once a node is popped out of the stack and is coloured GREY, we colour it BLACK.
\item Push all its neighbours which are not coloured BLACK.
\item Repeat steps 4 and 5 until the stack is empty. 
\end{DoxyEnumerate}

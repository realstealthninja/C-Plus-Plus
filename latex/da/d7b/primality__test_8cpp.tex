\doxysubsection{Detailed Description}
\href{https://en.wikipedia.org/wiki/Primality_test}{\texttt{ Primality test}} implementation. 

A simple and efficient implementation of a function to test if a number is prime, based on the fact that \begin{quote}
Every Prime number, except 2 and 3, are of the form $6k\pm1$ for integer values of k. This gives a 3x speed improvement. \end{quote}

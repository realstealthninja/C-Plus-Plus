\doxysubsection{Detailed Description}
Implementation of the \href{https://en.wikipedia.org/wiki/Karatsuba_algorithm}{\texttt{ Karatsuba algorithm for fast multiplication}} 

Given two strings in binary notation we want to multiply them and return the value. Simple approach is to multiply bits one by one which will give the time complexity of around O(n\texorpdfstring{$^\wedge$}{\string^}2). To make it more efficient we will be using Karatsuba algorithm to find the product which will solve the problem O(nlogn) of time. \begin{DoxyAuthor}{Author}
\href{https://github.com/Swastyy}{\texttt{ Swastika Gupta}} 

\href{https://github.com/poypoyan}{\texttt{ Ameer Carlo Lubang}} 
\end{DoxyAuthor}

\doxysubsection{Detailed Description}
\mbox{[}Travelling Salesman Problem\mbox{]} (\href{https://en.wikipedia.org/wiki/Travelling_salesman_problem}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Travelling\+\_\+salesman\+\_\+problem}}) implementation 

\begin{DoxyAuthor}{Author}
\href{http://github.com/Mayank17M}{\texttt{ Mayank Mamgain}}
\end{DoxyAuthor}
Travelling salesman problem asks\+: Given a list of cities and the distances between each pair of cities, what is the shortest possible route that visits each city exactly once and returns to the origin city? TSP can be modeled as an undirected weighted graph, such that cities are the graph\textquotesingle{}s vertices, paths are the graph\textquotesingle{}s edges, and a path\textquotesingle{}s distance is the edge\textquotesingle{}s weight. It is a minimization problem starting and finishing at a specified vertex after having visited each other vertex exactly once. This is the naive implementation of the problem.

\doxysubsection{Detailed Description}
Computes N\texorpdfstring{$^\wedge$}{\string^}th Fibonacci number given as input argument. Uses custom build arbitrary integers library to perform additions and other operations. 

Took 0.\+608246 seconds to compute 50,000\texorpdfstring{$^\wedge$}{\string^}th Fibonacci number that contains 10450 digits!

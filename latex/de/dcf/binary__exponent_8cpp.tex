\doxysubsection{Detailed Description}
C++ Program to find Binary Exponent Iteratively and Recursively. 

Calculate $a^b$ in $O(\log(b))$ by converting $b$ to a binary number. Binary exponentiation is also known as exponentiation by squaring. \begin{DoxyNote}{Note}
This is a far better approach compared to naive method which provide $O(b)$ operations.
\end{DoxyNote}
Example\+: 10 in base 2 is 1010.       \begin{eqnarray*}
2^{10_d} &=& 2^{1010_b} = 2^8 * 2^2\\
2^1 &=& 2\\
2^2 &=& (2^1)^2 = 2^2 = 4\\
2^4 &=& (2^2)^2 = 4^2 = 16\\
2^8 &=& (2^4)^2 = 16^2 = 256\\
\end{eqnarray*} Hence to calculate 2\texorpdfstring{$^\wedge$}{\string^}10 we only need to multiply $2^8$ and $2^2$ skipping $2^1$ and $2^4$.

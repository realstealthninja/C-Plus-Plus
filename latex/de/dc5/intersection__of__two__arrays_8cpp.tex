\doxysubsection{Detailed Description}
Implementation for the \href{https://en.wikipedia.org/wiki/Intersection_(set_theory)}{\texttt{ Intersection of two sorted Arrays}} algorithm. 

The intersection of two arrays is the collection of all the elements that are common in both the first and second arrays. This implementation uses ordered arrays, and an algorithm to correctly order them and return the result as a new array (vector). \begin{DoxySeeAlso}{See also}

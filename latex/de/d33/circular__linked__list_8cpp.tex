\doxysubsection{Detailed Description}
Implementation for a \href{https://www.geeksforgeeks.org/circular-linked-list/}{\texttt{ Circular Linked List}}. 

A Circular Linked List is a variation on the regular linked list, in which the last node has a pointer to the first node, which creates a full circle. Consequently, this allows any node to be used as the starting point for the list. \begin{DoxyAuthor}{Author}

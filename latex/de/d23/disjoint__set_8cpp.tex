A disjoint set data structure (also called union find or merge find set) is a data structure that tracks a set of elements partitioned into a number of disjoint (non-\/overlapping) subsets. Some situations where disjoint sets can be used are-\/ to find connected components of a graph, kruskal\textquotesingle{}s algorithm for finding \doxylink{namespace_minimum}{Minimum} Spanning Tree etc. There are two operation which we perform on disjoint sets -\/ 1) Union 2) Find

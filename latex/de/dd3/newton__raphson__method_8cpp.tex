\doxysubsection{Detailed Description}
Solve the equation $f(x)=0$ using \href{https://en.wikipedia.org/wiki/Newton\%27s_method}{\texttt{ Newton-\/\+Raphson method}} for both real and complex solutions. 

The $(i+1)^\text{th}$ approximation is given by\+:   \[
x_{i+1} = x_i - \frac{f(x_i)}{f\textnormal{\textquotesingle}(x_i)}
\]

\doxysubsection{Detailed Description}
Implements \href{https://www.codesdope.com/blog/article/backtracking-to-
solve-a-rat-in-a-maze-c-java-pytho/}{\texttt{ Rat in a Maze}} algorithm. 

A Maze is given as N\texorpdfstring{$\ast$}{*}N binary matrix of blocks where source block is the upper left most block i.\+e., maze\mbox{[}0\mbox{]}\mbox{[}0\mbox{]} and destination block is lower rightmost block i.\+e., maze\mbox{[}N-\/1\mbox{]}\mbox{[}N-\/1\mbox{]}. A rat starts from source and has to reach destination. The rat can move only in two directions\+: forward and down. In the maze matrix, 0 means the block is dead end and 1 means the block can be used in the path from source to destination.

\doxysubsection{Detailed Description}
Implementation of the \href{https://www.geeksforgeeks.org/wildcard-pattern-matching/}{\texttt{ Wildcard Matching}} problem. 

Given a matching string and a pattern, implement wildcard pattern matching with support for {\ttfamily ?} and {\ttfamily \texorpdfstring{$\ast$}{*}}. {\ttfamily ?} matches any single character. {\ttfamily \texorpdfstring{$\ast$}{*}} matches any sequence of characters (including the empty sequence). The matching should cover the entire matching string (not partial). The task is to determine if the pattern matches with the matching string \begin{DoxyAuthor}{Author}
\href{https://github.com/Swastyy}{\texttt{ Swastika Gupta}} 
\end{DoxyAuthor}

\doxysubsection{Detailed Description}
\href{https://en.wikipedia.org/wiki/Ternary_search}{\texttt{ Ternary search}} algorithm 

This is a divide and conquer algorithm. It does this by dividing the search space by 3 parts and using its property (usually monotonic property) to find the desired index.


\begin{DoxyItemize}
\item Time Complexity \+: O(log3 n)
\item Space Complexity \+: O(1) (without the array) 
\end{DoxyItemize}

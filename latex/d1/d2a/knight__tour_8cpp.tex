\doxysubsection{Detailed Description}
\href{https://en.wikipedia.org/wiki/Knight\%27s_tour}{\texttt{ Knight\textquotesingle{}s tour}} algorithm 

A knight\textquotesingle{}s tour is a sequence of moves of a knight on a chessboard such that the knight visits every square only once. If the knight ends on a square that is one knight\textquotesingle{}s move from the beginning square (so that it could tour the board again immediately, following the same path, the tour is closed; otherwise, it is open.

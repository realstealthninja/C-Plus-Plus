\doxysubsection{Detailed Description}
Implementation of the \href{https://www.sanfoundry.com/cpp-program-implement-quick-sort-using-randomisation}{\texttt{ Random Pivot Quick Sort}} algorithm. 


\begin{DoxyItemize}
\item A random pivot quick sort algorithm is pretty much same as quick sort with a difference of having a logic of selecting next pivot element from the input array.
\item Where in quick sort is fast, but still can give you the time complexity of O(n\texorpdfstring{$^\wedge$}{\string^}2) in worst case.
\item To avoid hitting the time complexity of O(n\texorpdfstring{$^\wedge$}{\string^}2), we use the logic of randomize the selection process of pivot element. \begin{DoxyVerb}     ### Logic
         * The logic is pretty simple, the only change is in the
\end{DoxyVerb}
 partitioning algorithm, which is selecting the pivot element.
\begin{DoxyItemize}
\item Instead of selecting the last or the first element from array for pivot we use a random index to select pivot element.
\item This avoids hitting the O(n\texorpdfstring{$^\wedge$}{\string^}2) time complexity in practical use cases. \begin{DoxyVerb}  ### Partition Logic
      * Partitions are done such as numbers lower than the "pivot"
\end{DoxyVerb}
 element is arranged on the left side of the "{}pivot"{}, and number larger than the "{}pivot"{} element are arranged on the right part of the array. \begin{DoxyVerb}  ### Algorithm
      * Select the pivot element randomly using getRandomIndex() function
\end{DoxyVerb}
 from this namespace.
\end{DoxyItemize}
\end{DoxyItemize}

Initialize the p\+Ind (partition index) from the start of the array.
\begin{DoxyItemize}
\item Loop through the array from start to less than end. (from start to \texorpdfstring{$<$}{<} end). (Inside the loop) \+:-\/
\begin{DoxyItemize}
\item Check if the current element (arr\mbox{[}i\mbox{]}) is less than the pivot element in each iteration.
\item If current element in the iteration is less than the pivot element, then swap the elements at current index (i) and partition index (p\+Ind) and increment the partition index by one.
\end{DoxyItemize}
\item At the end of the loop, swap the pivot element with partition index element.
\item Return the partition index from the function.
\end{DoxyItemize}

\doxysubsection{Detailed Description}
\href{https://en.wikipedia.org/wiki/Prefix_sum}{\texttt{ Prefix Sum Array}} data structure implementation. 

Prefix Sum Array is a data structure, that allows answering sum in some range queries. It can answer most sum range queries in O(1), with a build time complexity of O(\+N). But it hasn\textquotesingle{}t an update querie.


\begin{DoxyItemize}
\item Running Time Complexity ~\newline

\item Build \+: O(\+N) ~\newline

\item Range Query \+: O(1) ~\newline

\doxysubsection{Detailed Description}
Faster computation for $a^b$. 

Program that computes $a^b$ in $O(logN)$ time. It is based on formula that\+:
\begin{DoxyEnumerate}
\item if $b$ is even\+: $a^b = a^\frac{b}{2} \cdot a^\frac{b}{2} = {a^\frac{b}{2}}^2$
\item if $b$ is odd\+:  $a^b = a^\frac{b-1}{2}
 \cdot a^\frac{b-1}{2} \cdot a = {a^\frac{b-1}{2}}^2 \cdot a$
\end{DoxyEnumerate}

We can compute $a^b$ recursively using above algorithm.

\begin{DoxyItemize}
\item A better version of bubble sort algorithm
\item Bubble sort compares adjacent values whereas comb sort uses gap larger than 1
\item Best case Time complexity O(n) Worst case Time complexity O(n\texorpdfstring{$^\wedge$}{\string^}2)

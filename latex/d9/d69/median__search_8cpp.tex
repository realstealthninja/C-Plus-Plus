\doxysubsection{Detailed Description}
Implementation of \href{https://en.wikipedia.org/wiki/Median_of_medians}{\texttt{ Median search}} algorithm. @cases from \href{https://brilliant.org/wiki/median-finding-algorithm/}{\texttt{ here}} 

Given an array A\mbox{[}1,...,n\mbox{]} of n numbers and an index i, where 1 ≤ i ≤ n, find the i-\/th smallest element of A. median\+\_\+of\+\_\+medians(\+A, i)\+: \#divide A into sublists of len 5 sublists = \mbox{[}A\mbox{[}j\+:j+5\mbox{]} for j in range(0, len(\+A), 5)\mbox{]} medians = \mbox{[}sorted(sublist)\mbox{[}len(sublist)/2\mbox{]} for sublist in sublists\mbox{]} if len(medians) \texorpdfstring{$<$}{<}= 5\+: pivot = sorted(medians)\mbox{[}len(medians)/2\mbox{]} else\+: \#the pivot is the median of the medians pivot = median\+\_\+of\+\_\+medians(medians, len(medians)/2) \#partitioning step low = \mbox{[}j for j in A if j \texorpdfstring{$<$}{<} pivot\mbox{]} high = \mbox{[}j for j in A if j \texorpdfstring{$>$}{>} pivot\mbox{]} k = len(low) if i \texorpdfstring{$<$}{<} k\+: return median\+\_\+of\+\_\+medians(low,i) elif i \texorpdfstring{$>$}{>} k\+: return median\+\_\+of\+\_\+medians(high,i-\/k-\/1) else\+: \#pivot = k return pivot

\begin{DoxyNote}{Note}
this algorithm implements median search for only arrays which have distinct elements
\end{DoxyNote}
Here are some example lists you can use to see how the algorithm works A = \mbox{[}1,2,3,4,5,1000,8,9,99\mbox{]} (Contain Unique Elements) B = \mbox{[}1,2,3,4,5,6\mbox{]} (Contains Unique Elements) print median\+\_\+of\+\_\+medians(\+A, 0) \#should be 1 print median\+\_\+of\+\_\+medians(\+A,7) \#should be 99 print median\+\_\+of\+\_\+medians(\+B,4) \#should be 5

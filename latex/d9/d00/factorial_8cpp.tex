\doxysubsection{Detailed Description}
Find the \href{https://en.wikipedia.org/wiki/Factorial}{\texttt{ factorial}} of a given number. 

Calculate factorial via recursion  \[n! = n\times(n-1)\times(n-2)\times(n-3)\times\ldots\times3\times2\times1
      = n\times(n-1)!\] for example\+: $5! = 5\times4! = 5\times4\times3\times2\times1 = 120$

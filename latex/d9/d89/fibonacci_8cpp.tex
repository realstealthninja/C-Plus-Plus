n-\/th \href{https://en.wikipedia.org/wiki/Fibonacci_sequence}{\texttt{ Fibonacci number}}. 

Naive recursive implementation to calculate the n-\/th Fibonacci number. \[\text{fib}(n) = \text{fib}(n-1) + \text{fib}(n-2)\]

\doxysubsection{Detailed Description}
\href{https://leetcode.com/problems/jump-game/}{\texttt{ Jumping Game}} algorithm implementation 

Given an array of non-\/negative integers, you are initially positioned at the first index of the array. Each element in the array represents your maximum jump length at that position. Determine if you are able to reach the last index. This solution takes in input as a vector and output as a boolean to check if you can reach the last position. We name the indices good and bad based on whether we can reach the destination if we start at that position. We initialize the last index as last\+Pos. Here, we start from the end of the array and check if we can ever reach the first index. We check if the sum of the index and the maximum jump count given is greater than or equal to the last\+Pos. If yes, then that is the last position you can reach starting from the back. After the end of the loop, if we reach the last\+Pos as 0, then the destination can be reached from the start position.

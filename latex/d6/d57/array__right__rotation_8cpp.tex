\doxysubsection{Detailed Description}
Implementation for the \href{https://www.javatpoint.com/program-to-right-rotate-the-elements-of-an-array}{\texttt{ Array right Rotation}} algorithm. 

Shifting an array to the right involves moving each element of the array so that it occupies a position of a certain shift value after its current one. This implementation uses a result vector and does not mutate the input. \begin{DoxySeeAlso}{See also}

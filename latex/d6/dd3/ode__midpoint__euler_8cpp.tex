\doxysubsection{Detailed Description}
Solve a multivariable first order \href{https://en.wikipedia.org/wiki/Ordinary_differential_equation}{\texttt{ ordinary differential equation (ODEs)}} using \href{https://en.wikipedia.org/wiki/Midpoint_method}{\texttt{ midpoint Euler method}} 

\begin{DoxyAuthor}{Authors}
\href{https://github.com/kvedala}{\texttt{ Krishna Vedala}} 
\end{DoxyAuthor}
The ODE being solved is\+:      \begin{eqnarray*}
   \dot{u} &=& v\\
   \dot{v} &=& -\omega^2 u\\
   \omega &=& 1\\
   [x_0, u_0, v_0] &=& [0,1,0]\qquad\ldots\text{(initial values)}
   \end{eqnarray*} The exact solution for the above problem is\+:    \begin{eqnarray*}
   u(x) &=& \cos(x)\\
   v(x) &=& -\sin(x)\\
   \end{eqnarray*} The computation results are stored to a text file {\ttfamily midpoint\+\_\+euler.\+csv} and the exact soltuion results in {\ttfamily exact.\+csv} for comparison. 

To implement \href{https://en.wikipedia.org/wiki/Van_der_Pol_oscillator}{\texttt{ Van der Pol oscillator}}, change the \doxylink{ode__midpoint__euler_8cpp_abaeae8f62a018d197f0187a1c80a90fe}{problem} function to\+:

\doxysubsection{Detailed Description}
Simple C++ implementation of the \href{https://en.wikipedia.org/wiki/MD5}{\texttt{ MD5 Hashing Algorithm}} 

\begin{DoxyAuthor}{Author}
\href{https://github.com/tGautot}{\texttt{ t\+Gautot}}
\end{DoxyAuthor}
The \href{https://en.wikipedia.org/wiki/MD5}{\texttt{ MD5 Algorithm}} is a hashing algorithm which was designed in 1991 by \href{https://en.wikipedia.org/wiki/Ron_Rivest}{\texttt{ Ronal Rivest}}.

\doxylink{namespace_m_d5}{MD5} is one of the most used hashing algorithm there is. Some of its use cases are\+:
\begin{DoxyEnumerate}
\item Providing checksum for downloaded software
\item Store salted password
\end{DoxyEnumerate}

However \doxylink{namespace_m_d5}{MD5} has be know to be cryptographically weak for quite some time, yet it is still widely used. This weakness was exploited by the \href{https://en.wikipedia.org/wiki/Flame_(malware)}{\texttt{ Flame Malware}} in 2012\hypertarget{md5_8cpp_autotoc_md82}{}\doxysubsubsubsection{\texorpdfstring{Algorithm}{Algorithm}}\label{md5_8cpp_autotoc_md82}
First of all, all values are expected to be in \mbox{[}little endian\mbox{]} (\href{https://en.wikipedia.org/wiki/Endianness}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Endianness}}). This is especially important when using part of the bytestring as an integer.

The first step of the algorithm is to pad the message for its length to be a multiple of 64 (bytes). This is done by first adding 0x80 (10000000) and then only zeroes until the last 8 bytes must be filled, where then the 64 bit size of the input will be added

Once this is done, the algo breaks down this padded message into 64 bytes chunks. Each chunk is used for one {\itshape round}, a round breaks the chunk into 16 blocks of 4 bytes. During these rounds the algorithm will update its 128 bit state (represented by 4 ints\+: A,B,C,D) For more precisions on these operations please see the \href{https://en.wikipedia.org/wiki/MD5\#Algorithm}{\texttt{ Wikipedia aritcle}}. The signature given by \doxylink{namespace_m_d5}{MD5} is its 128 bit state once all rounds are done. \begin{DoxyNote}{Note}
This is a simple implementation for a byte string but some implmenetations can work on bytestream, messages of unknown length. 
\end{DoxyNote}

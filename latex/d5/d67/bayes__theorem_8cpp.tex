\doxysubsection{Detailed Description}
\href{https://en.wikipedia.org/wiki/Bayes\%27_theorem}{\texttt{ Bayes\textquotesingle{} theorem}} 

Bayes\textquotesingle{} theorem allows one to find $P(A|B)$ given $P(B|A)$ or $P(B|A)$ given $P(A|B)$ and $P(A)$ and $P(B)$.~\newline
Note that $P(A|B)$ is read \textquotesingle{}The probability of A given that the event B has occured\textquotesingle{}.

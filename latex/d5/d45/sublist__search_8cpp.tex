\doxysubsection{Detailed Description}
Implementation of the \href{https://www.geeksforgeeks.org/sublist-search-search-a-linked-list-in-another-list}{\texttt{ Sublist Search Algorithm}} 

\hypertarget{sublist__search_8cpp_autotoc_md107}{}\doxysubsubsubsection{\texorpdfstring{Algorithm}{Algorithm}}\label{sublist__search_8cpp_autotoc_md107}

\begin{DoxyItemize}
\item Sublist search is used to detect a presence of one list in another list.
\item Suppose we have a single-\/node list (let\textquotesingle{}s say the first list), and we want to ensure that the list is present in another list (let\textquotesingle{}s say the second list), then we can perform the sublist search to find it.
\item For instance, the first list contains these elements\+: 23 -\/\texorpdfstring{$>$}{>} 30 -\/\texorpdfstring{$>$}{>} 41, and the second list contains these elements\+: 10 -\/\texorpdfstring{$>$}{>} 15 -\/\texorpdfstring{$>$}{>} 23 -\/\texorpdfstring{$>$}{>} 30 -\/\texorpdfstring{$>$}{>} 41 -\/\texorpdfstring{$>$}{>} 49. At a glance, we see that the first list presents in the second list.
\end{DoxyItemize}\hypertarget{sublist__search_8cpp_autotoc_md108}{}\doxysubsubsubsection{\texorpdfstring{Working}{Working}}\label{sublist__search_8cpp_autotoc_md108}

\begin{DoxyItemize}
\item The sublist search algorithm works by comparing the first element of the first list with the first element of the second list.
\item If the two values don\textquotesingle{}t match, it goes to the next element of the second list. It does this until the two values match.
\end{DoxyItemize}

\doxysubsection{Detailed Description}
Algorithm to find largest x such that p\texorpdfstring{$^\wedge$}{\string^}x divides n! (factorial) using Legendre\textquotesingle{}s Formula. 

Given an integer n and a prime number p, the task is to find the largest x such that p\texorpdfstring{$^\wedge$}{\string^}x (p raised to power x) divides n! (factorial).

\doxysubsection{Detailed Description}
Implementation of \href{https://en.wikipedia.org/wiki/Bogosort}{\texttt{ Bogosort algorithm}} 

In computer science, bogosort (also known as permutation sort, stupid sort, slowsort, shotgun sort, random sort, monkey sort, bobosort or shuffle sort) is a highly inefficient sorting algorithm based on the generate and test paradigm. Two versions of this algorithm exist\+: a deterministic version that enumerates all permutations until it hits a sorted one, and a randomized version that randomly permutes its input.\+Randomized version is implemented here.\hypertarget{bogo__sort_8cpp_autotoc_md109}{}\doxysubsubsubsection{\texorpdfstring{Algorithm}{Algorithm}}\label{bogo__sort_8cpp_autotoc_md109}
Shuffle the array untill array is sorted.

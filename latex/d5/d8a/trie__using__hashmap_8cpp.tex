\doxysubsection{Detailed Description}
Implementation of \href{https://en.wikipedia.org/wiki/Trie}{\texttt{ Trie}} data structure using Hash\+Map for different characters and method for predicting words based on prefix. 

\begin{DoxyAuthor}{Author}
\href{https://github.com/bharath000}{\texttt{ Venkata Bharath}}
\end{DoxyAuthor}
The \doxylink{class_trie}{Trie} data structure is implemented using unordered map to use memory optimally, predict\+\_\+words method is developed to recommend words based on a given prefix along with other methods insert, delete, search, startwith in trie. \begin{DoxySeeAlso}{See also}

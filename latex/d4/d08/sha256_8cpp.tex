\doxysubsection{Detailed Description}
Simple C++ implementation of the \mbox{[}SHA-\/256 Hashing Algorithm\mbox{]} (\href{https://en.wikipedia.org/wiki/SHA-2}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+SHA-\/2}}) 

\begin{DoxyAuthor}{Author}
\href{https://github.com/mdanisulh}{\texttt{ Md. Anisul Haque}} 
\end{DoxyAuthor}
\href{https://en.wikipedia.org/wiki/SHA-2}{\texttt{ SHA-\/2}} is a set of cryptographic hash functions that was designed by the \href{https://en.wikipedia.org/wiki/National_Security_Agency}{\texttt{ NSA}} and first published in 2001. SHA-\/256 is a part of the SHA-\/2 family. SHA-\/256 is widely used for authenticating software packages and secure password hashing.

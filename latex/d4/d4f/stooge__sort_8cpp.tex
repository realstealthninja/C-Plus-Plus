\doxysubsection{Detailed Description}
\href{https://en.wikipedia.org/wiki/Stooge_sort}{\texttt{ Stooge sort implementation}} in C++ 

Stooge sort is a recursive sorting algorithm. It divides the array into 3 parts and proceeds to\+:
\begin{DoxyItemize}
\item sort first two thirds of the array
\item sort last two thirds of the array
\item sort first two thirds of the array It\textquotesingle{}s time complexity is O(n\texorpdfstring{$^\wedge$}{\string^}(log3/log1.\+5)), which is about O(n\texorpdfstring{$^\wedge$}{\string^}2.7), which makes it to be not the most efficient sorting algorithm on the street on average. Space complexity is O(1). 
\end{DoxyItemize}

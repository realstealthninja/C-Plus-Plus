\doxysubsection{Detailed Description}
Implementation to \mbox{[}Travelling Salesman problem using bit-\/masking\mbox{]} (\href{https://www.geeksforgeeks.org/travelling-salesman-problem-set-1/}{\texttt{ https\+://www.\+geeksforgeeks.\+org/travelling-\/salesman-\/problem-\/set-\/1/}}) 

Given the distance/cost(as and adjacency matrix) between each city/node to the other city/node , the problem is to find the shortest possible route that visits every city exactly once and returns to the starting point or we can say the minimum cost of whole tour.

Explanation\+: INPUT -\/\texorpdfstring{$>$}{>} You are given with a adjacency matrix A = \{\} which contains the distance between two cities/node.

OUTPUT -\/\texorpdfstring{$>$}{>} \doxylink{namespace_minimum}{Minimum} cost of whole tour from starting point

\doxysubsection{Detailed Description}
\mbox{[}Borůvkas Algorithm\mbox{]}(\href{https://en.wikipedia.org/wiki/Bor\%C5\%AFvka's_algorithm}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Borůvka\textquotesingle{}s\+\_\+algorithm}}) to find the \doxylink{namespace_minimum}{Minimum} Spanning Tree 

\begin{DoxyAuthor}{Author}
\href{https://github.com/JNardoni}{\texttt{ Jason Nardoni}}
\end{DoxyAuthor}


Boruvka\textquotesingle{}s algorithm is a greepy algorithm to find the MST by starting with small trees, and combining them to build bigger ones.
\begin{DoxyEnumerate}
\item Creates a group for every vertex.
\item looks through each edge of every vertex for the smallest weight. Keeps track of the smallest edge for each of the current groups.
\end{DoxyEnumerate}

Combine each group with the group it shares its smallest edge, adding the smallest edge to the MST.
\begin{DoxyEnumerate}
\item Repeat step 2-\/3 until all vertices are combined into a single group.
\end{DoxyEnumerate}

It assumes that the graph is connected. Non-\/connected edges can be represented using 0 or INT\+\_\+\+MAX

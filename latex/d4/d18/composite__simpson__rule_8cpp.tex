\doxysubsection{Detailed Description}
Implementation of the Composite Simpson Rule for the approximation. 

The following is an implementation of the Composite Simpson Rule for the approximation of definite integrals. More info -\/\texorpdfstring{$>$}{>} wiki\+: \href{https://en.wikipedia.org/wiki/Simpson\%27s_rule\#Composite_Simpson's_rule}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Simpson\%27s\+\_\+rule\#\+Composite\+\_\+\+Simpson\textquotesingle{}s\+\_\+rule}}

The idea is to split the interval in an EVEN number N of intervals and use as interpolation points the xi for which it applies that xi = x0 + i\texorpdfstring{$\ast$}{*}h, where h is a step defined as h = (b-\/a)/N where a and b are the first and last points of the interval of the integration \mbox{[}a, b\mbox{]}.

We create a table of the xi and their corresponding f(xi) values and we evaluate the integral by the formula\+: I = h/3 \texorpdfstring{$\ast$}{*} \{f(x0) + 4\texorpdfstring{$\ast$}{*}f(x1) + 2\texorpdfstring{$\ast$}{*}f(x2) + ... + 2\texorpdfstring{$\ast$}{*}f(x\+N-\/2) + 4\texorpdfstring{$\ast$}{*}f(x\+N-\/1) + f(x\+N)\}

That means that the first and last indexed i f(xi) are multiplied by 1, the odd indexed f(xi) by 4 and the even by 2.

In this program there are 4 sample test functions f, g, k, l that are evaluated in the same interval.

Arguments can be passed as parameters from the command line argv\mbox{[}1\mbox{]} = N, argv\mbox{[}2\mbox{]} = a, argv\mbox{[}3\mbox{]} = b

N must be even number and a\texorpdfstring{$<$}{<}b.

In the end of the \doxylink{generate__parentheses_8cpp_gae66f6b31b5ad750f1fe042a706a4e3d4}{main()} i compare the program\textquotesingle{}s result with the one from mathematical software with 2 decimal points margin.

\doxysubsection{Detailed Description}
An implementation for finding the \href{https://www.youtube.com/watch?v=5cPbNCrdotA}{\texttt{ Inorder successor of a binary search tree}} Inorder successor of a node is the next node in Inorder traversal of the Binary Tree. Inorder Successor is NULL for the last node in Inorder traversal. 

\hypertarget{inorder__successor__of__bst_8cpp_autotoc_md88}{}\doxysubsubsubsection{\texorpdfstring{Case 1\+: The given node has the right node/subtree}{Case 1\+: The given node has the right node/subtree}}\label{inorder__successor__of__bst_8cpp_autotoc_md88}
\begin{DoxyVerb} * In this case, the left-most deepest node in the right subtree will
\end{DoxyVerb}
 come just after the given node as we go to left deep in inorder.
\begin{DoxyItemize}
\item Go deep to left most node in right subtree. OR, we can also say in case if BST, find the minimum of the subtree for a given node.
\end{DoxyItemize}\hypertarget{inorder__successor__of__bst_8cpp_autotoc_md89}{}\doxysubsubsubsection{\texorpdfstring{Case 2\+: The given node does not have a right node/subtree}{Case 2\+: The given node does not have a right node/subtree}}\label{inorder__successor__of__bst_8cpp_autotoc_md89}
\hypertarget{inorder__successor__of__bst_8cpp_autotoc_md90}{}\doxysubsubsubsubsection{\texorpdfstring{Method 1\+: Use parent pointer (store the address of parent nodes)}{Method 1\+: Use parent pointer (store the address of parent nodes)}}\label{inorder__successor__of__bst_8cpp_autotoc_md90}

\begin{DoxyItemize}
\item If a node does not have the right subtree, and we already visited the node itself, then the next node will be its parent node according to inorder traversal, and if we are going to parent from left, then the parent would be unvisited.
\item In other words, go to the nearest ancestor for which given node would be in left subtree.
\end{DoxyItemize}\hypertarget{inorder__successor__of__bst_8cpp_autotoc_md91}{}\doxysubsubsubsubsection{\texorpdfstring{Method 2\+: Search from the root node}{Method 2\+: Search from the root node}}\label{inorder__successor__of__bst_8cpp_autotoc_md91}

\begin{DoxyItemize}
\item In case if there is no link from a child node to the parent node, we need to walk down the tree starting from the root node to the given node, by doing so, we are visiting every ancestor of the given node.
\item In order successor would be the deepest node in this path for which given node is in left subtree.
\end{DoxyItemize}

\doxysubsection{Detailed Description}
Implementation for the \href{https://www.geeksforgeeks.org/reverse-tree-path/}{\texttt{ Reversing a Binary Tree}} recursively algorithm. 

A binary tree can be reversed by swapping the left and right child of a node at each node, starting from the root, and cascading below. This solution aims to provide an implementation of a recursive reversal of a binary tree. \begin{DoxyAuthor}{Author}
\href{https://github.com/polarvoid}{\texttt{ Alvin}} 
\end{DoxyAuthor}

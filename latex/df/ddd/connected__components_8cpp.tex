\mbox{[}\doxylink{class_graph}{Graph} Connected Components (Connected Components)\mbox{]} (\href{https://en.wikipedia.org/wiki/Component_(graph_theory)}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Component\+\_\+(graph\+\_\+theory)}})

\end{DoxyAuthor}
A graph is a collection of nodes also called vertices and these vertices are connected by edges. A connected component in a graph refers to a set of vertices which are reachable form one another.


\begin{DoxyPre}
Example -\/ Here is graph with 3 connected components

     1   4           5               8
    / \textbackslash{} /           / \textbackslash{}             / \textbackslash{}
   2-\/-\/-\/3           6   7           9   10

   first          second           third
   component      component        component
\end{DoxyPre}

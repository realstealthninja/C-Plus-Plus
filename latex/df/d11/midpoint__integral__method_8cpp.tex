\doxysubsection{Detailed Description}
A numerical method for easy \href{https://en.wikipedia.org/wiki/Midpoint_method}{\texttt{ approximation of integrals}} 

The idea is to split the interval into N of intervals and use as interpolation points the xi for which it applies that xi = x0 + i\texorpdfstring{$\ast$}{*}h, where h is a step defined as h = (b-\/a)/N where a and b are the first and last points of the interval of the integration \mbox{[}a, b\mbox{]}.

We create a table of the xi and their corresponding f(xi) values and we evaluate the integral by the formula\+: I = h \texorpdfstring{$\ast$}{*} \{f(x0+h/2) + f(x1+h/2) + ... + f(x\+N-\/1+h/2)\}

Arguments can be passed as parameters from the command line argv\mbox{[}1\mbox{]} = N, argv\mbox{[}2\mbox{]} = a, argv\mbox{[}3\mbox{]} = b. In this case if the default values N=16, a=1, b=3 are changed then the tests/assert are disabled.

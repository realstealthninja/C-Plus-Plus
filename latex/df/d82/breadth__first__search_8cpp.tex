Breadth First Search also quoted as BFS is a \doxylink{class_graph}{Graph} Traversal Algorithm. Time Complexity O(\texorpdfstring{$\vert$}{|}\+V\texorpdfstring{$\vert$}{|} + \texorpdfstring{$\vert$}{|}\+E\texorpdfstring{$\vert$}{|}) where V are the number of vertices and E are the number of edges in the graph.

Applications of Breadth First Search are


\begin{DoxyEnumerate}
\item Finding shortest path between two vertices say u and v, with path length measured by number of edges (an advantage over depth first search algorithm)
\item Ford-\/\+Fulkerson Method for computing the maximum flow in a flow network.
\item Testing bipartiteness of a graph.
\item Cheney\textquotesingle{}s Algorithm, Copying garbage collection.
\end{DoxyEnumerate}

And there are many more...

\doxysubsubsubsubsection*{working}

In the implementation below we first created a graph using the adjacency list representation of graph. Breadth First Search Works as follows it requires a vertex as a start vertex, Start vertex is that vertex from where you want to start traversing the graph. We maintain a bool array or a vector to keep track of the vertices which we have visited so that we do not traverse the visited vertices again and again and eventually fall into an infinite loop. Along with this boolen array we use a \doxylink{struct_queue}{Queue}.


\begin{DoxyEnumerate}
\item First we mark the start vertex as visited.
\item Push this visited vertex in the \doxylink{struct_queue}{Queue}.
\item while the queue is not empty we repeat the following steps
\begin{DoxyEnumerate}
\item Take out an element from the front of queue
\item Explore the adjacency list of this vertex if element in the adjacency list is not visited then we push that element into the queue and mark this as visited 
\end{DoxyEnumerate}
\end{DoxyEnumerate}

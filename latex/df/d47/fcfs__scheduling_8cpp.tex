\doxysubsection{Detailed Description}
Implementation of \doxylink{class_f_c_f_s}{FCFS} CPU scheduling algorithm. 

\doxylink{class_f_c_f_s}{FCFS} is a non-\/preemptive CPU scheduling algorithm in which whichever process arrives first, gets executed first. If two or more processes arrive simultaneously, the process with smaller process ID gets executed first. \doxylink{}{https\+://bit.\+ly/3\+ABNXOC}Pratyush Vatsa

\doxysubsection{Detailed Description}
\href{https://en.wikipedia.org/wiki/Exponential_search}{\texttt{ Exponential search algorithm}} 

\begin{DoxyCopyright}{Copyright}
2020 Divide-\/et-\/impera-\/11
\end{DoxyCopyright}
The algorithm try to search the range where the key should be. If it has been found we do a binary search there. The range of the search grows by exponential every time. If the key is larger than the last element of array, the start of block(block\+\_\+front) will be equal to the end of block(block\+\_\+size) and the algorithm return null ponter, every other cases the algoritm return fom the loop.

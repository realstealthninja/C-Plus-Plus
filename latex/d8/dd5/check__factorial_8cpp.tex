\doxysubsection{Detailed Description}
A simple program to check if the given number is a \href{https://en.wikipedia.org/wiki/Factorial}{\texttt{ factorial}} of some number or not. 

A factorial number is the sum of k! where any value of k is a positive integer. \href{https://www.mathsisfun.com/numbers/factorial.html}{\texttt{ https\+://www.\+mathsisfun.\+com/numbers/factorial.\+html}}

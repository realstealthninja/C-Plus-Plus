\doxysubsection{Detailed Description}
Implementation of Linear \mbox{[}\doxylink{struct_queue}{Queue} using array\mbox{]} (\href{https://www.geeksforgeeks.org/array-implementation-of-queue-simple/}{\texttt{ https\+://www.\+geeksforgeeks.\+org/array-\/implementation-\/of-\/queue-\/simple/}}). 

The Linear \doxylink{struct_queue}{Queue} is a data structure used for holding a sequence of values, which can be added to the end line (enqueue), removed from head of line (dequeue) and displayed. \hypertarget{queue__using__array_8cpp_autotoc_md43}{}\doxysubsubsubsection{\texorpdfstring{Algorithm}{Algorithm}}\label{queue__using__array_8cpp_autotoc_md43}
Values can be added by increasing the {\ttfamily rear} variable by 1 (which points to the end of the array), then assigning new value to {\ttfamily rear}\textquotesingle{}s element of the array.

Values can be removed by increasing the {\ttfamily front} variable by 1 (which points to the first of the array), so it cannot reached any more.

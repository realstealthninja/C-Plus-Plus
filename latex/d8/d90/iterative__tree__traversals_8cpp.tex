\hypertarget{iterative__tree__traversals_8cpp_autotoc_md92}{}\doxysubsubsubsection{\texorpdfstring{Iterative Preorder Traversal of a tree}{Iterative Preorder Traversal of a tree}}\label{iterative__tree__traversals_8cpp_autotoc_md92}
Create a Stack that will store the \doxylink{struct_node}{Node} of Tree. Push the root node into the stack. Save the root into the variabe named as current, and pop and elemnt from the stack. Store the data of current into the result array, and start traversing from it. Push both the child node of the current node into the stack, first right child then left child. Repeat the same set of steps untill the Stack becomes empty. And return the result array as the preorder traversal of a tree.\hypertarget{iterative__tree__traversals_8cpp_autotoc_md93}{}\doxysubsubsubsection{\texorpdfstring{Iterative Postorder Traversal of a tree}{Iterative Postorder Traversal of a tree}}\label{iterative__tree__traversals_8cpp_autotoc_md93}
Create a Stack that will store the \doxylink{struct_node}{Node} of Tree. Push the root node into the stack. Save the root into the variabe named as current, and pop and elemnt from the stack. Store the data of current into the result array, and start traversing from it. Push both the child node of the current node into the stack, first left child then right child. Repeat the same set of steps untill the Stack becomes empty. Now reverse the result array and then return it to the calling function as a postorder traversal of a tree.\hypertarget{iterative__tree__traversals_8cpp_autotoc_md94}{}\doxysubsubsubsection{\texorpdfstring{Iterative Inorder Traversal of a tree}{Iterative Inorder Traversal of a tree}}\label{iterative__tree__traversals_8cpp_autotoc_md94}
Create a Stack that will store the \doxylink{struct_node}{Node} of Tree. Push the root node into the stack. Save the root into the variabe named as current. Now iterate and take the current to the extreme left of the tree by traversing only to its left. Pop the elemnt from the stack and assign it to the current. Store the data of current into the result array. Repeat the same set of steps until the Stack becomes empty or the current becomes NULL. And return the result array as the inorder traversal of a tree.

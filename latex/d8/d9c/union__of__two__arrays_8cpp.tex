\doxysubsection{Detailed Description}
Implementation for the \href{https://en.wikipedia.org/wiki/Union_(set_theory)}{\texttt{ Union of two sorted Arrays}} algorithm. 

The Union of two arrays is the collection of all the unique elements in the first array, combined with all of the unique elements of a second array. This implementation uses ordered arrays, and an algorithm to correctly order them and return the result as a new array (vector). \begin{DoxySeeAlso}{See also}

\doxysubsection{Detailed Description}
Given a linked list L\mbox{[}0,....,n\mbox{]} of n numbers, find the middle node. 

The technique utilized in this implementation is the \mbox{[}"{}\+Floyd\textquotesingle{}s tortoise and hare"{}\mbox{]}(\href{https://en.wikipedia.org/wiki/Cycle_detection\#Floyd's_tortoise_and_hare}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Cycle\+\_\+detection\#\+Floyd\textquotesingle{}s\+\_\+tortoise\+\_\+and\+\_\+hare}}) approach. This technique uses two pointers that iterate through the list at different \textquotesingle{}speeds\textquotesingle{} in order to solve problems. In this implementation, for every iteration the slow pointer advances one node while the fast pointer advances two nodes. The result of this is that since the fast pointer moves twice as fast as the slow pointer, when the fast pointer reaches the end of the list the slow pointer will be pointing to the middle node of the list.

Here are some example lists you can use to see how the algorithm works A = \mbox{[}1,2,3,4,5\mbox{]} B = \mbox{[}1,2,3,4,5,6\mbox{]} print median(\+A) \#should be 39 print median(\+B) \#should be 4

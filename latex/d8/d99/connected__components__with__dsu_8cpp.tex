\doxysubsection{Detailed Description}
\href{https://en.wikipedia.org/wiki/Disjoint_union}{\texttt{ Disjoint union}} 

The Disjoint union is the technique to find connected component in graph efficiently.\hypertarget{connected__components__with__dsu_8cpp_autotoc_md75}{}\doxysubsubsubsection{\texorpdfstring{Algorithm}{Algorithm}}\label{connected__components__with__dsu_8cpp_autotoc_md75}
In \doxylink{class_graph}{Graph}, if you have to find out the number of connected components, there are 2 options
\begin{DoxyEnumerate}
\item Depth first search
\item Disjoint union 1st option is inefficient, Disjoint union is the most optimal way to find this.
\end{DoxyEnumerate}

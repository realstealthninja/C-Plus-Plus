\doxysubsection{Detailed Description}
Algorithm of \href{https://en.wikipedia.org/wiki/Radix_sort}{\texttt{ Radix sort}} 

\begin{DoxyAuthor}{Author}
\href{https://github.com/Suyashjaiswal}{\texttt{ Suyash Jaiswal}}
\end{DoxyAuthor}
Sort the vector of unsigned integers using radix sort i.\+e. sorting digit by digit using \href{https://en.wikipedia.org/wiki/Counting_sort}{\texttt{ Counting Sort}} as subroutine. Running time of radix sort is O(d\texorpdfstring{$\ast$}{*}(n+b)) where b is the base for representing numbers and d in the max digits in input integers and n is number of unsigned integers. consider example for n = 5, aray elements = 432,234,143,332,123 sorting digit by digit sorting according to 1) 1st digit place =\texorpdfstring{$>$}{>} 432, 332, 143, 123, 234

2) 2nd digit place =\texorpdfstring{$>$}{>} 123, 432, 332, 234, 143

3) 3rd digit place =\texorpdfstring{$>$}{>} 123, 143, 234, 332, 432

using count sort at each step, which is stable. stable =\texorpdfstring{$>$}{>} already sorted according to previous digits.

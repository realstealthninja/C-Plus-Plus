\doxysubsection{Detailed Description}
Compute statistics for data entered in rreal-\/time. 

This algorithm is really beneficial to compute statistics on data read in realtime. For example, devices reading biometrics data. The algorithm is simple enough to be easily implemented in an embedded system. \begin{DoxyAuthor}{Author}
\href{https://github.com/kvedala}{\texttt{ Krishna Vedala}}

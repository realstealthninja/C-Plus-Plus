\doxysubsection{Detailed Description}
A balanced binary search tree (BST) on the basis of binary search tree and heap\+: the \href{https://en.wikipedia.org/wiki/Treap}{\texttt{ Treap}} algorithm implementation. 

Implementation of the treap data structre

Support operations including insert, erase, and query (the rank of specified element or the element ranked x) as the same as BST

But these operations take O(log N) time, since treap keeps property of heap using rotate operation, and the desired depth of the tree is O(log N). There\textquotesingle{}s very little chance that it will degenerate into a chain like BST

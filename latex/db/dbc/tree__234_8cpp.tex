\doxysubsection{Detailed Description}
A demo 2-\/3-\/4 tree implementation. 

2–3–4 tree is a self-\/balancing data structure that is an isometry of red–black trees. Though we seldom use them in practice, we study them to understand the theory behind Red-\/\+Black tree. Please read following links for more infomation. \href{https://en.wikipedia.org/wiki/2\%E2\%80\%933\%E2\%80\%934_tree}{\texttt{ 2–3–4 tree}} \href{https://www.educative.io/page/5689413791121408/80001}{\texttt{ 2-\/3-\/4 Trees\+: A Visual Introduction}} We Only implement some basic and complicated operations in this demo. Other operations should be easy to be added. \begin{DoxyAuthor}{Author}
\href{https://github.com/fedom}{\texttt{ liuhuan}} 
\end{DoxyAuthor}

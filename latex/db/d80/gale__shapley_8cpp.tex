\doxysubsection{Detailed Description}
\href{https://en.wikipedia.org/wiki/Gale\%E2\%80\%93Shapley_algorithm}{\texttt{ Gale Shapley Algorithm}} 

This implementation utilizes the Gale-\/\+Shapley algorithm to find stable matches.

{\bfseries{Gale Shapley Algorithm}} aims to find a stable matching between two equally sized sets of elements given an ordinal preference for each element. The algorithm was introduced by David Gale and Lloyd Shapley in 1962.

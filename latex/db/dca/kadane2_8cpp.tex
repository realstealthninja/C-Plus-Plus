\doxysubsection{Detailed Description}
Implementation of \href{https://en.wikipedia.org/wiki/Kadane\%27s_algorithm}{\texttt{ Kadane Algorithm}} 

Kadane algorithm is used to find the maximum sum subarray in an array and maximum sum subarray problem is the task of finding a contiguous subarray with the largest sum\hypertarget{kadane2_8cpp_autotoc_md72}{}\doxysubsubsubsection{\texorpdfstring{Algorithm}{Algorithm}}\label{kadane2_8cpp_autotoc_md72}
The simple idea of the algorithm is to search for all positive contiguous segments of the array and keep track of maximum sum contiguous segment among all positive segments(curr\+\_\+sum is used for this) Each time we get a positive sum we compare it with max\+\_\+sum and update max\+\_\+sum if it is greater than curr\+\_\+sum

\doxysubsection{Detailed Description}
Implementation of \mbox{[}0-\/1 \doxylink{namespace_knapsack}{Knapsack} Problem\mbox{]} (\href{https://en.wikipedia.org/wiki/Knapsack_problem}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Knapsack\+\_\+problem}}) 

Given weights and values of n items, put these items in a knapsack of capacity {\ttfamily W} to get the maximum total value in the knapsack. In other words, given two integer arrays {\ttfamily val\mbox{[}0..n-\/1\mbox{]}} and {\ttfamily wt\mbox{[}0..n-\/1\mbox{]}} which represent values and weights associated with n items respectively. Also given an integer W which represents knapsack capacity, find out the maximum value subset of {\ttfamily val\mbox{[}\mbox{]}} such that sum of the weights of this subset is smaller than or equal to W. You cannot break an item, either pick the complete item or donâ€™t pick it (0-\/1 property)\hypertarget{0__1__knapsack_8cpp_autotoc_md69}{}\doxysubsubsubsection{\texorpdfstring{Algorithm}{Algorithm}}\label{0__1__knapsack_8cpp_autotoc_md69}
The idea is to consider all subsets of items and calculate the total weight and value of all subsets. Consider the only subsets whose total weight is smaller than {\ttfamily W}. From all such subsets, pick the maximum value subset.

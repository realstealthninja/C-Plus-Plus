\doxysubsection{Detailed Description}
Iterative implementation of \href{https://en.wikipedia.org/wiki/Factorial}{\texttt{ Factorial}} 

\begin{DoxyAuthor}{Author}
\href{https://github.com/Renjian-buchai}{\texttt{ Renjian-\/buchai}}
\end{DoxyAuthor}
Calculates factorial iteratively.  \[n! = n\times(n-1)\times(n-2)\times(n-3)\times\ldots\times3\times2\times1
      = n\times(n-1)!\] for example\+: $4! = 4\times3! = 4\times3\times2\times1 = 24$

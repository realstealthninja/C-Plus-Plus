\doxysubsection{Detailed Description}
Find real extrema of a univariate real function in a given interval using \href{https://en.wikipedia.org/wiki/Brent\%27s_method}{\texttt{ Brent\textquotesingle{}s method}}. 

Refer the algorithm discoverer\textquotesingle{}s publication \href{https://maths-people.anu.edu.au/~brent/pd/rpb011i.pdf}{\texttt{ online}} and also associated book\+: \begin{quote}
R. P. Brent, Algorithms for Minimization without Derivatives, Prentice-\/\+Hall, Englewood Cliffs, New Jersey, 1973 \end{quote}

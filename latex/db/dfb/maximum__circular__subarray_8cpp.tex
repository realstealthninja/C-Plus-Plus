\doxysubsection{Detailed Description}
C++ program for maximum contiguous circular sum problem using \href{https://en.wikipedia.org/wiki/Maximum_subarray_problem}{\texttt{ Kadane\textquotesingle{}s Algorithm}} 

The idea is to modify Kadaneâ€™s algorithm to find a minimum contiguous subarray sum and the maximum contiguous subarray sum, then check for the maximum value between the max\+\_\+value and the value left after subtracting min\+\_\+value from the total sum. For more information, check \href{https://www.geeksforgeeks.org/maximum-contiguous-circular-sum/}{\texttt{ Geeks For Geeks}} explanation page.

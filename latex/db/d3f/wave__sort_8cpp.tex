\doxysubsection{Detailed Description}
Implementation of the \href{https://www.geeksforgeeks.org/sort-array-wave-form-2/}{\texttt{ Wave sort}} algorithm. 

Wave Sort is a sorting algorithm that works in $O(nlogn)$ time assuming the sort function used works in $O(nlogn)$ time. \begin{DoxyAuthor}{Author}

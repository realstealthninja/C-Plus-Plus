5The problem can be solved with DP but constraints are high.
6<br/>\f$a_i = b_i\f$ (for \f$i <= k\f$)
7<br/>\f$a_i = c_1 a_{i-1} + c_2 a_{i-2} + ... + c_k a_{i-k}\f$ (for \f$i > k\f$)
8<br/>Taking the example of Fibonacci series, \f$k=2\f$
9<br/>\f$b_1 = 1,\; b_2=1\f$
10<br/>\f$c_1 = 1,\; c_2=1\f$
11<br/>\f$a = \begin{bmatrix}0& 1& 1& 2& \ldots\end{bmatrix}\f$
12<br/>This way you can find the \f$10^{18}\f$ fibonacci number%MOD.
13I have given a general way to use it. The program takes the input of B and C
14matrix.
15
16Steps for Matrix Expo
171. Create vector F1 : which is the copy of B.
182. Create transpose matrix (Learn more about it on the internet)
193. Perform \f$T^{n-1}\f$ [transpose matrix to the power n-1]
204. Multiply with F to get the last matrix of size (1\f$\times\f$k).

22The first element of this matrix is the required result.

70/** computing integer power of a matrix using recursive multiplication.
71 * @note A must be a square matrix for this algorithm.
72 * \param [in] A base matrix
73 * \param [in] p exponent
74 * \return matrix of same dimension as A
75 */
76vector<vector<ll>> power(const vector<vector<ll>> &A, ll p) {
77 if (p == 1)
78 return A;
79 if (p % 2 == 1) {
80 return multiply(A, power(A, p - 1));
81 } else {
82 vector<vector<ll>> X = power(A, p / 2);
83 return multiply(X, X);
84 }
